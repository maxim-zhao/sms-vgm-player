 file to the player for it to
 work. You can do this by using
 this commandline:

 copy /b vgmplayer.stub+
         vgmfile.vgm output.sms

      (all on one line)
 where:
 ± vgmplayer.stub is the 16KB
   player file
 ± vgmfile.vgm is an
   UNCOMPRESSED VGM file
 ± output.sms is the combined
   file which will be created
